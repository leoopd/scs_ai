      Wie stelle ich die Netzwerk-Firewall bei meinem Root Server (Windows und Linux) ein?

      Richtlinien

      Mit Hilfe von Firewall-Richtlinien können Sie bestimmte Arten von Netzwerkverkehr zulassen. Im Bereich Netzwerk  --> Firewall-Richtlinien werden alle von Ihnen konfigurierten Firewall-Richtlinien in einer Tabelle aufgelistet.



      Sie können einem Server eine automatische oder selbst erstellte Firewall-Richtlinie zuweisen. Dazu gehen Sie wie folgt vor:

      Klicken Sie links im Menü auf Infrastruktur --> Server und dann noch einmal auf den entsprechenden Server.
      Nun werden Ihnen die Server-Eigenschaften angezeigt.

      Anschließend klicken Sie unten bei den Firewall-Richtlinien auf das Schreibsymbol der entsprechenden Firewall-Richtlinie und anschließend auf Änderungen speichern.

      Die Konfiguration des Servers kann einige Minuten dauern.

      Angaben zu den einzelnen Firewall-Richtlinien

      Im Folgenden finden Sie die wichtigsten Angaben zu den einzelnen Firewall-Richtlinien:
      Name: Name, der der Firewall-Richtlinie zugewiesen wurde. Sie können den Namen jederzeit ändern.
      Status: Aktueller Status der Firewall-Richtlinie
      •   Aktiv: Die Firewall-Richtlinie ist aktiviert.
      •   Deaktiviert: Die Firewall-Richtlinie ist deaktiviert.
      •   Wird konfiguriert: Die Konfiguration der Firewall-Richtlinie wird geändert.
      •   Wird entfernt: Die Firewall-Richtlinie wird gelöscht.
      Port: Port(s), für die die Firewall-Richtlinie gilt.

      Um alle Informationen zur gewünschten Firewall-Richtlinie aufzurufen, aktivieren Sie die gewünschte Firewall-Richtlinie. Anschließend werden die Firewall-Regeln für den eingehenden Datenverkehr in einer Tabelle angezeigt. Die Tabelle enthält die folgenden, zusätzlichen Informationen:

      Aktion: Legt fest, dass der Datenverkehr zugelassen wird, der in der Firewall-Regel definiert wird.
      Erlaubte IP: IP-Adresse(n), für die der eingehende Datenverkehr erlaubt wird.
      Protokoll: Protokoll, über das die Datenverbindung aufgebaut wird. 
      Port(s): Port, der für die Datenverbindung verwendet wird.
      Beschreibung: Beschreibung der Firewall-Regel.
      Erstellungsdatum: Datum, an dem die Firewall-Richtlinie erstellt wurde.
      Zugewiesene IP: Server, für die die Firewall-Richtlinie gilt.
      Verlauf: Im Verlauf werden alle bisher durchgeführten Aktionen durchgeführt.

      Wenn Sie auf den Button Standardwerte einfügen klicken, öffnet sich eine Auswahl an Standard-Firewall-Regeln, die Sie zur schnellen Konfiguration Ihrer Firewall-Richtlinie nutzen können.

      Firewall-Richtlinie erstellen

      So erstellen Sie eine Firewall-Richtlinie, um bestimmte Arten von Netzwerkverkehr zuzulassen:
      Klicken Sie zunächst im Menü links auf Netzwerk --> Firewall-Richtlinien und dann auf Erstellen.
      Geben Sie nun den gewünschten Namen für die Firewall-Richtlinie ein.

      Im Bereich Konfiguration können Sie individuelle oder vordefinierte Firewall-Regeln erstellen.

      Individuelle Firewall-Regel

      • Erlaubte IP: Geben Sie die gewünschte IP-Adresse ein, für die die Firewall-Regel erstellt werden soll. Alternativ können Sie einen gewünschten IP-Adressbereich im folgenden Format eingeben: 217.12.1.5-217.12.1.128
      • Protokoll: Wählen Sie das gewünschte Protokoll aus.

      Kurzdefinition der Protokolle:

      TCP

      UDP

      TCP/UDP

      ICMP

      IPSEC

      GRE

      ANY
      • Port(s): Geben Sie den gewünschten Port ein. Alternativ können Sie einen Portbereich oder mehrere Ports eintragen. 
      Erklärung Portranges: TBD
      Um eine Regel für einen Portbereich zu erstellen, geben Sie den unteren und den oberen Bereichswert ein und trennen diese Werte mit einem Bindestrich: z.B.: 4000-5000. Um für mehrere Ports jeweils eine separate Firewall-Regel zu erstellen, geben Sie die gewünschten Ports mit einem Komma an, um diese zu trennen.



      Vordefinierte Firewall-Regel

      Um eine vordefinierte Regel hinzuzufügen, klicken Sie aufden Button Standardwerte einfügen. Wählen Sie anschließend die von Ihnen gewünschte Regel und passen Sie diese ggf. manuell an. Im Folgenden sind einige der verfügbaren Regeln aufgeführt:

      • Alle Ports blockieren: Blockiert alle eingehenden Ports des Servers.
      • Alle Ports öffnen: Öffnet alle eingehenden Ports des Servers.
      • HTTP: Öffnet den Port 80 für eingehende Anfragen.
      • HTTPS: Öffnet den Port 443 für eingehende Anfragen.
      • POP3 SSL: Öffnet den Port 995 für eingehende Anfragen.
      • IMAP4 SSL: Öffnet den Port 993 für eingehende Anfragen.

      Um eine neue Firewall-Regel zu speichern, klicken Sie auf das Plussymbol .

      Um eine Firewall-Regel zurückzusetzen, klicken Sie auf das x-Symbol .

      Um eine Firewall-Regel zu löschen, klicken Sie auf Minussymbol .

      Aus Sicherheitsgründen ist der SMTP-Port 25 (ausgehend) gesperrt. Wenn Sie den SMTP-Port freischalten möchten, wenden Sie sich bitte an den Support.


      Um die Firewall-Richtlinie zu erstellen, klicken Sie auf Erstellen.
      Sobald die Firewall-Richtlinie aktiv ist, können Sie einen Server hinzufügen.

      Firewall-Richtlinie einem Server zuweisen

      Sie können einem Server eine automatisch erstellte oder eine selbst erstellte Firewall-Richtlinie zuweisen.

      Bitte gehen Sie wie folgt vor:
      Klicken Sie zunächst links im Menü auf Infrastruktur und dann auf den Server. Nun werden Ihnen die Server-Eigenschaften werden angezeigt.

      Als nächstes klicken Sie unten bei den Firewall-Richtlinien auf das Symbol mit dem Stift und wählen die entsprechende Firewall-Richtlinie aus. Anschließend speichern Sie die Änderungen.

      Die Konfiguration des Servers kann einige Minuten in Anspruch nehmen.

      Firewall-Richtlinie duplizieren

      Wenn Sie eine bestehende Firewall-Richtlinie duplizieren wollen, werden automatisch alle Einstellungen übernommen. Dadurch entfällt die erneute Eingabe aller Ports.

      Sobald Sie sich im Server-Cloud-Panel eingeloggt haben, klicken Sie links im Menü unter Netzwerk auf Firewall-Richtlinien und klicken noch mal auf den Server.
      Wählen Sie aus der Liste eine Firewall-Richtlinie aus und klicken Sie oben auf den Button Duplizieren.
      BEstätigen Sie die Sicherheitsmeldung mit Ja und anschließend wird die Firewall-Richtlinie dupliziert.

      Firewall-Richtlinie löschen

      So löschen Sie eine Firewall-Richtlinie:
      Klicken Sie links im Menü auf Netzwerk --> Firewall-Richtlinien und in der Liste auf die entsprechende Richtlinie, die sie löschen wollen. Nachdem Sie auf den Button Löschen gedrück haben, bestätigen Sie die Sicherheitsabfrage mit Ja.



      Die Firewall-Richtlinie wird nun gelöscht.
