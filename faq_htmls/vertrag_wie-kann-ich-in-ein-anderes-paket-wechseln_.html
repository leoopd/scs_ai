      Wie kann ich in ein anderes Paket wechseln?

      Sie benötigen mehr Leistung und Features oder ist Ihnen das derzeitige Paket inhaltlich zu groß?
      Ganz nach unserem Motto #faireinfacht ermöglichen wir Ihnen jederzeit ein Wechsel in ein anderes Paket. Dabei spielt es keine Rolle, ob Sie ein Upgrade oder ein Downgrade durchführen möchten.

      In diesem Artikel erfahren Sie, wie Sie Ihr Paket (Webhosting, Homepage-Baukasten, Webshop oder HiDrive) wechseln können und welche Punkte es dabei zu beachten gilt.

      Wie kann ich in ein größeres Paket wechseln?



       

      Mit einem Upgrade können Sie mehr Leistung und neue Funktionen für Ihr bestehendes STRATO Paket freischalten. Alternativ haben Sie die Möglichkeit Ihr Paket in ein anderes Produkt zu wandeln. Wie schnell und einfach dies möglich ist, werden Sie in dieser Anleitung Schritt für Schritt erfahren.

      Wenn Sie ein Upgrade beantragen möchten, loggen Sie sich bitte in den STRATO Kunden-Login ein. Wählen Sie dort bitte das gewünschte Paket aus für welches Sie ein Upgrade durchführen möchten. Klicken Sie bitte auf Paket upgraden.

      Im ersten Schritt gelangen Sie zur Paketauswahl, an dieser Stelle werden Ihnen alle möglichen Pakete angezeigt, auf die ein Upgrade Ihres derzeitigen Paketes möglich ist. Durch einen Klick auf den entsprechenden Reiter können Sie zwischen den Kategorien wählen.

      Haben Sie ein Paket ausgewählt, wird Ihnen eine Aufstellung der Produktfeatures für das bisherige und das zukünftige Paket angezeigt. Mit der Klick auf alle anzeigen wird Ihnen eine detaillierte Auflistung der Features zur Verfügung gestellt.

      Nach der Auswahl des gewünschten Paketes gelangen Sie über die Schaltfläche Weiter zum nächsten Schritt.

      In der Bestellübersicht erhalten Sie eine Übersicht über den Wandel und die entstehenden Kosten für Ihr neues Paket.

      Nach Prüfung der Angaben in der Bestellübersicht können Sie die Bestellung abschließen.

      Akzeptieren Sie dazu bitte die Allgemeinen Geschäftsbedingungen und klicken Sie auf die Schaltfläche Jetzt kostenpflichtig bestellen. Beachten Sie, dass das Upgrade erst nach dem Klick auf diese Schaltfläche beauftragt wird. Mit Umstellung des Paketes gelten die Vertragsbedingungen des neu gewählten Paketes.

      Wie kann ich in ein kleineres Paket wechseln?


      Sie können bei STRATO jederzeit auf ein kleineres Paket wechseln (d. h. downgraden). Den Paketwechsel nehmen wir gegen eine Gebühr in Höhe von 20,- EUR für Sie vor. Es gibt, je nach gewähltem Paket, Unterschiede hinsichtlich der Umsetzung des Paketwechsels.

      Die Umstellung Ihres Auftrages können Sie auch direkt und kostenlos über Ihren STRATO Kunden-Login vornehmen. Die Option zur Umstellung Ihres Paketes finden Sie unter dem Menüpunkt Mein Vertrag / Vertragsänderung / Paketwechsel (Downgrade).

      Achtung: Sollten Sie Inhaber eines V-Servers sein, können Sie kein Downgrade vornehmen, da dies mit Datenverlust einhergehen könnte.

      Wählen Sie im nächsten Schritt bitte das Paket aus, für welches Sie ein Downgrade durchführen möchten.

      Sie haben nun die Möglichkeit den Zeitpunkt für den Paketwechsel zwischen „zum Vertragsende“ oder „zu sofort“ auszuwählen.

      Ein sofortiges Downgrade ist nur innerhalb der gleichen Produktkategorie möglich.

      Ebenfalls erhalten Sie einen Überblick auf welches Paket Sie ein Downgrade durchführen können. Durch einen Klick auf den Punkt alle anzeigen erhalten Sie eine Übersicht über die Leistungen des aktuellen und zukünftigen Paketes.

      Nachdem Sie Ihr gewünschtes Paket ausgewählt haben, gelangen Sie über den Punkt Weiter mit diesem Tarif in den nächsten Schritt.

      Sie werden nun über alle vertraglichen Besonderheiten, wie Zeitpunkt der Umstellung, Kosten Laufzeit etc., informiert. Ebenfalls müssen Sie in diesem Schritt unsere allgemeinen Geschäftsbedingungen und die entstehenden Einschränkungen aufgrund von Leistungswegfall bestätigen.

      Der Paketwechsel wird mit der Eingabe Ihres Kunden-Login Passworts und dem Klick auf Kostenpflichtig bestellen final bestätigt.

      Welche Kosten kommen auf mich zu?

      Ein Paketwechselbei der STRATO AG ist grundsätzlich kostenfrei. Wir berücksichtigen selbstverständlich von Ihnen im Voraus bezahlte Beträge bei der Abbuchung für Ihr zukünftiges Paket. Überbezahlte Zeiträume werden wir mit nächster Rechnungslegung erstatten.

      Wenn Sie einen Downgrade vornehmen möchten, können wir dies für Sie gerne am Telefon nach Authentifizierung mit der vierstelligen Service-Pin durchführen, dabei fallen einmalig 20,- EUR pro Downgrade an.


      Was passiert mit meinen Domains nach einem Paketwechsel?

      Ihre Domains werden bei einem Paketwechsel übernommen und bleiben erhalten. Sie brauchen also keine Domains zu kündigen, sondern können alle behalten.

      Beachten Sie aber bitte, das bisher kostenfreie Inklusivdomains zu kostenpflichtigen Zusatzdomains werden können, da sich in Ihrem zukünftigem Paket gegebenenfalls auch die Anzahl der kostenfreien Inklusivdomains verringert. Die Kosten für eine zusätzliche Domain richtet sich hierbei nach den gewählten Topleveldomains. Eine Preisübersicht erhalten Sie hier.


      Wie lange dauert die Umstellung meines Paketes?

      Die Umstellung Ihres Paketes dauert bei einem Upgrade im Regelfall einen Arbeitstag. Bei einem Downgrade haben Sie die Möglichkeit ein Wechsel zu sofort oder zum Vertragsende zu erfassen.

      Sobald der Paketwechsel vollzogen ist, können Sie die neuen Möglichkeiten des Paketes voll nutzen. Wir informieren Sie nach Abschluss der Umstellung per E-Mail an die von Ihnen angegebene E-Mail Adresse.

      Kann ich nachträglich auf ein anderes LivePages-Paket umsteigen?

      Nein, dies ist nicht mehr möglich. Wir stellen Ihnen in diesem Fall unseren neuen STRATO Homepage-Baukasten zur Verfügung.
