      So einfach konfigurieren Sie den STRATO Windows SMTP-Mail Server


      Hintergrundwissen: Da der POP3-Server auf den vorhandenen SMTP-Server aufsetzt, nimmt der SMTP-Dienst auch erstmal alle E-Mails entgegen, unabhängig davon, ob es hierfür eine Mailbox gibt oder nicht. Wird eine E-Mail entgegengenommen, für die es kein Postfach gibt, so wird diese E-Mail gelöscht und der Absender erhält eine Information über die Nichtzustellbarkeit der E-Mail.

      Die Postfächer für jeden Benutzer werden vom POP3-Dienst zur Verfügung gestellt.
      So einfach konfigurieren Sie den Windows POP3-Mail Server

      Da E-Mails von beliebigen Servern kommen können, werden die STRATO Windows Server standardmäßig so zur Verfügung gestellt, dass der SMTP-Dienst keine Verbindungen von externen Quellen erlaubt. Sofern dies von Ihnen jedoch nicht erwünscht ist, müssen Sie diese Einstellung folgendermaßen ändern.



      Nachdem Sie sich auf Ihrem STRATO Windows-Server eingeloggt haben, öffnen Sie die IIS-Verwaltung:
      Start > Programme > Verwaltung > Internetinformationsdienste-Manager:
      Öffnen Sie anschließend die Eigenschaften für den [Default SMTP Virtual Server] (rechter Mausklick oder über den Menüpunkt Aktion > Eigenschaften).

      Klicken Sie auf den Karteikartenreiter Zugriff und anschließend auf Verbindung

       
      Sie können jetzt festlegen, welche Computer Zugriff auf Ihren Server haben sollen.
      Dies bezieht sich jedoch nur auf die Verbindung generell, nicht aber auf E-Mails, FTP, o.ä.)

      Wichtig: Aus Sicherheitsgründen empfehlen wir Ihnen, die Einstellungen für Weitergabe zu überprüfen.

      Damit nicht Dritte unter Umständen die Möglichkeit haben, Nachrichten über Ihren SMTP-Server zu versenden (z.B. SPAM), sollte hier nur localhost mit der Adresse 127.0.0.1 eingetragen sein.

      Sollte hier kein Eintrag vorhanden sein, klicken Sie auf den Button Hinzufügen, geben anschließend die IP [127.0.0.1] ein und bestätigen die Eingabe mit OK.


      Hinweis:
      Andere Rechner die über den SMTP-Server E-Mails verschicken wollen, müssen sich bei diesem authentifizieren. Dafür muss bei Authentifizierung das Kontrollkästchen Standardauthentifizierung angehakt sein und Benutzername und Passwort müssen festgelegt werden.

      Auf dem Rechner der über den SMTP-Server E-Mails versenden möchte, müssen dann dieser Benutzername, Passwort und SMTP-Serveradresse bei den E-Mail-Kontoeinstellungen angegeben werden.
